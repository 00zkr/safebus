# SafeBus Project Archive

This folder is the documentation workspace for archiving the development of the SafeBus project.

SafeBus is a school transport tracking demo for Toubkal IT Center. It contains:

- A backend REST and realtime API built with Node.js, Express, MySQL, JWT, and Socket.IO.
- A web dashboard built with React and Vite for the administrator role.
- A native Android mobile app built with Kotlin for parent and driver roles.
- A Docker Compose environment for MySQL, backend, and web services.

## Files in this folder

- `PROJECT_DETAILS.md`: project overview, architecture, modules, roles, features, and technical stack.
- `DEVELOPMENT_ARCHIVE_STEPS.md`: complete steps to document, archive, package, verify, and present the development work.
- `DOCUMENTS_TO_PREPARE.md`: other useful documents and papers you can create later, including README, PowerPoint, LaTeX reports, test reports, and deployment papers.
